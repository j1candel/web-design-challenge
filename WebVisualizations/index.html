Summary: Latitude vs. X

This project was made to find trends between climate elements such as wind speed, humidity, cloudiness and temperature with changes in latitude and longitude. To attempt to find possible trends,we made 500 random latitude and longitude and by using the city api it gave us the nearest city to those cordinates

After finding 500 random cities, we used the weather api to find the humidity, temperature, cloudiness and wind speed. This website will allow the visitor to visually see the trends
